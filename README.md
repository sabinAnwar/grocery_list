# Grocery Bud

Simple grocery list app built with HTML, CSS, and JavaScript.

## Features

- Add grocery items to the list.
- Edit existing items.
- Delete individual items.
- Clear the full list.
- Save items in `localStorage` so they stay after refresh.

## Enhancement Idea

One useful improvement would be to prevent duplicate items from being added. For example, if the user enters an item that already exists in the list, the app could ignore it or show a message instead of adding it again.

This would make the list cleaner and avoid repeated entries.
